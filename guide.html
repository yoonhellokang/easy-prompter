텔레프롬프터 상세 사용 가이드

대본 준비부터 장비 배치, 속도 설정과 촬영 전 점검까지 순서대로 살펴보세요.

1. 대본 준비

홈의 텔레프롬프터에서 ‘대본 편집’을 누른 뒤 기존 예문을 지우고 원고를 붙여 넣습니다. 편집 완료 버튼이나 Esc를 누르면 저장됩니다. 문장이 너무 길면 의미 단위로 줄을 나누고, 호흡할 지점에는 빈 줄을 넣으세요.

2. 읽기 설정

속도

처음에는 낮은 속도로 20~30초를 읽어 봅니다. 글을 기다리는 시간이 길면 조금 높이고, 문장 끝을 자주 놓치면 낮추세요. 위·아래 방향키로 재생 중에도 조절할 수 있습니다.

글자 크기와 행간

눈을 가늘게 뜨지 않고 읽을 수 있는 크기가 적당합니다. 화면과 거리가 멀수록 크게 설정하고, 다음 줄을 잘못 읽는다면 행간을 넓히세요. 한 화면에 많은 글을 담는 것보다 시선을 안정적으로 유지하는 것이 중요합니다.

정렬과 좌우 반전

가운데 정렬은 짧은 문장에, 왼쪽 정렬은 긴 설명문에 유용합니다. 좌우 반전은 반사 유리가 있는 전용 장비에서만 켜고 화면을 직접 볼 때는 끕니다.

3. 카메라와 화면 배치

렌즈와 읽는 기준선의 거리를 최대한 좁히세요. 노트북을 사용할 때는 카메라 창을 대본 바로 위에 두거나 외장 카메라를 화면 중앙 가까이에 설치합니다. 스마트폰은 삼각대에 고정하고 렌즈 바로 아래에 대본이 흐르도록 배치하면 시선 움직임이 줄어듭니다.

4. 단축키

  • Space: 재생 또는 일시 정지
  • ↑ / ↓: 속도 한 단계 조절
  • Shift + ↑ / ↓: 속도 다섯 단계 조절
  • R: 대본 처음으로 이동
  • E: 대본 편집 시작
  • Esc 또는 Ctrl/Command + Enter: 편집 완료

5. 촬영 전 체크리스트

  • 방해 금지 모드를 켜 알림이 화면에 나타나지 않게 합니다.
  • 배터리와 저장 공간을 확인합니다.
  • 30초 테스트를 녹화해 눈동자 움직임과 음량을 확인합니다.
  • 대본의 첫 문장과 마지막 문장은 가능하면 외워 자연스럽게 시작하고 끝냅니다.
  • 중요한 촬영이라면 원본 대본을 별도 문서에도 보관합니다.

6. 문제 해결

대본이 저장되지 않아요

비공개 브라우징 또는 저장 공간을 차단하는 설정에서는 localStorage가 유지되지 않을 수 있습니다. 일반 브라우저 창에서 다시 시도하고 사이트 데이터 저장이 허용되어 있는지 확인하세요.

전체 화면이 열리지 않아요

브라우저는 반드시 사용자의 버튼 클릭 직후에만 전체 화면을 허용합니다. ‘전체 화면’ 버튼을 직접 누르세요. 일부 모바일 브라우저는 웹 전체 화면 기능을 제한합니다.

스크롤이 끊겨 보여요

다른 탭과 앱을 닫고 절전 모드를 해제해 보세요. 아주 큰 글자나 긴 대본에서는 기기 성능에 따라 차이가 날 수 있습니다.
