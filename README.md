# MediPrompter

MediPrompter는 브라우저에서 바로 실행되는 텔레프롬프터입니다.

`index.html` 파일을 열면 별도 설치 없이 사용할 수 있습니다.

## 주요 기능

- 대본 자동 스크롤
- 시작/정지 버튼 및 Space 단축키
- 속도, 글자 크기, 행간, 대본 너비 조절
- 슬라이더와 숫자 직접 입력 지원
- 왼쪽 정렬 / 가운데 정렬
- 화면에서 바로 대본 편집
- 좌우 반전
- 전체 화면
- 진행률 표시
- 읽는 기준선 20% 위치 표시
- 편집 중이 아닐 때 `← / →` 키로 위아래 이동
- 브라우저 localStorage 자동 저장

## 단축키

- `Space`: 시작 / 정지
- `↑ / ↓`: 속도 조절
- `Shift + ↑ / ↓`: 속도 크게 조절
- `← / →`: 대본 위아래 이동
- `Shift + ← / →`: 대본 크게 이동
- `R`: 처음으로 이동
- `E`: 대본 편집
- `Esc` 또는 `Ctrl + Enter`: 대본 편집 종료

## 파일 구성

- `index.html`: 최신 안정 버전입니다. 현재 `teleprompter-ver11.html` 기준입니다.
- `archive/`: 이전 작업 버전 보관 폴더입니다.

## 참고
