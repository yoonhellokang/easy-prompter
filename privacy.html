PRIVACY

개인정보처리방침

Ezprompter는 현재 회원가입, 계정 또는 자체 분석 서버를 운영하지 않습니다. 입력한 대본은 사용자의 브라우저 안에서만 처리됩니다.

1. 직접 수집하는 정보

사이트는 이름, 이메일 주소, 전화번호 등의 개인정보를 직접 요청하거나 수집하지 않습니다. 문의를 위해 외부 GitHub 서비스를 이용하면 해당 서비스의 개인정보처리방침이 적용됩니다.

2. 기기에 저장되는 정보

대본, 스크롤 속도, 글자 크기, 행간, 정렬 및 반전 설정을 편의 기능 제공 목적으로 localStorage에 저장합니다. 이 정보는 사이트 운영자에게 전송되지 않으며 사용자가 브라우저의 사이트 데이터 삭제 기능으로 언제든 제거할 수 있습니다.

3. 호스팅 과정의 기술 정보

사이트는 Cloudflare Pages를 통해 제공됩니다. 보안과 안정적인 전송을 위해 Cloudflare가 IP 주소, 요청 시각, 브라우저 정보 등의 기술 로그를 처리할 수 있습니다. 자세한 내용은 Cloudflare의 개인정보처리방침을 확인하세요.

4. 광고와 쿠키

현재 이 사이트에는 Google AdSense 광고 코드가 설치되어 있지 않습니다. 향후 광고를 도입할 경우 Google과 그 파트너가 쿠키를 사용해 광고를 제공하거나 측정할 수 있으며, 적용 전에 이 방침과 동의 절차를 관련 법령에 맞게 갱신합니다.

5. 데이터 삭제

브라우저 설정에서 easy-prompter.pages.dev의 사이트 데이터 또는 localStorage를 삭제하면 저장된 대본과 설정이 제거됩니다. 비공개 브라우징에서는 창을 닫을 때 삭제될 수 있습니다.

6. 아동의 개인정보

사이트는 연령을 확인하거나 아동의 개인정보를 의도적으로 수집하지 않습니다. 개인정보를 입력할 필요 없이 도구를 사용할 수 있습니다.

7. 변경 및 문의

기능이나 외부 서비스가 변경되면 이 문서를 갱신하고 날짜를 표시합니다. 문의는 GitHub Issues를 이용할 수 있으며, 공개 게시판에 개인정보를 작성하지 마세요.
