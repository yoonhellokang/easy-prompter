ABOUT

말하는 사람에게 집중하는 도구

Ezprompter는 발표와 촬영을 준비하는 사람이 설치나 가입 과정 없이 대본 연습을 시작할 수 있도록 만든 무료 웹 도구입니다.

왜 만들었나요?

간단한 촬영을 위해 무거운 프로그램을 설치하거나 계정을 만들 필요는 없어야 한다고 생각했습니다. 핵심 기능을 브라우저 표준 기술만으로 구현해 스마트폰, 태블릿, 노트북에서 같은 방식으로 쓸 수 있게 했습니다.

어떻게 작동하나요?

입력한 대본과 속도, 글자 크기 등의 설정은 현재 브라우저의 localStorage에 저장됩니다. 별도의 계정 서버나 대본 데이터베이스를 운영하지 않으므로 대본이 기기 밖으로 전송되지 않습니다. 브라우저 데이터를 지우면 저장된 대본도 함께 삭제됩니다.

운영 원칙

  • 핵심 텔레프롬프터 기능은 무료로 제공합니다.
  • 대본을 수집하거나 판매하지 않습니다.
  • 기능과 개인정보 처리 방식을 이해하기 쉬운 말로 알립니다.
  • 키보드와 작은 화면에서도 사용할 수 있도록 지속적으로 개선합니다.

문의와 개선 제안

오류 제보와 기능 제안은 GitHub Issues에서 받을 수 있습니다. 공개 저장소이므로 대본이나 개인정보는 게시하지 마세요.
